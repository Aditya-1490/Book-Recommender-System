# Book-Recommender-System
This project implements a Book Recommender System that suggests books based on popularity metrics and user similarity using collaborative filtering.



## Dataset
The dataset used in this project is too large to be uploaded to GitHub.

You can download it from:
- https://www.kaggle.com/datasets/arashnic/book-recommendation-dataset

After downloading, place the files in the project root:
- Books.csv
- Ratings.csv
- Users.csv
